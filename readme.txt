== Installation ==

1. Upload 'importwp-sportspress' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "ImportWP" and start importing.

Please make sure you have ImportWP installed before activating this plugin. For further documentation on installing and using ImportWP features can be found [here](https://www.importwp.com/documentation/).
